# 共享记忆索引

## 定位

本文是跨 agent 共享记忆责任域的母索引,只负责定位 Claude / Codex 都需要遵守的长期规则、工作区约定、后续提示和踩坑入口。具体规则、坑位根因和执行细节写入 `docs/共享记忆/` 下的中文子文档。

`CLAUDE.md` 与 `AGENTS.md` 是同级 agent 入口;二者需要共享的信息统一指向本文或本文索引到的子文档,不要在两个入口文件中复制长篇规则。

## 新增信息落点

- 跨 agent 都要遵守的入口层级、任务记录、共享记忆规则:写入 [协作入口规则](共享记忆/协作入口规则.md)。
- 长期有效的工作区结构、crate 依赖边界、构建运行命令和测试约定:写入 [工作区与构建约定](共享记忆/工作区与构建约定.md)。
- 尚未排入具体任务、但对后续实现有指导价值的路线提示:写入 [后续事项与路线提示](共享记忆/后续事项与路线提示.md)。
- 反复影响实现判断的坑位、根因和规避规则:写入 [踩坑记录](共享记忆/踩坑记录.md)。
- 具体架构事实写入 [系统架构索引](系统架构索引.md) 或 `docs/架构/`;具体任务流水写入 `TODO.md` 与 `docs/任务/`。
