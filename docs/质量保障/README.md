- **性能**:整体纪律良好;用户可感的「进入很慢」主因是**子 shell 首 prompt 的异步启动延迟**(非前台冻结;SSH spawn 立即返回——已更正初版误判),其次是整屏重绘类 agent 下终端**全量重建分配**(P1)。
