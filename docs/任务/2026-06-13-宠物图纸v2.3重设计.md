# 宠物图纸 v2.3 重设计(SHEET 05)

## 定位

按 [宠物系统规则](../宠物/宠物系统规则.md)(v2.3 陪伴版)重做设计图纸 [design/panels/05-pet-system.html](../../design/panels/05-pet-system.html),让图纸与现行规则一致,并补出规则要求的 **板 G(互动记忆)** 与 **板 H(磷光通道)** 审核稿。

## 背景

原 05 图纸停留在 ~v2.1:

- 板 B「品种架」与板 D 右键菜单仍含 **「随机刷新」** —— 已被 v2.1 反规则废除(持续性老虎机瓦解身份连续性)。
- 缺规则 D 性格系统、规则 C 活物引擎、规则 E 工作共情、规则 G/H/I 互动记忆、规则 J 磷光通道。
- v2.3 任务「后续 2」要求先出磷光通道审核稿(覆盖幽灵终端/Quick Look/浮层/分屏),图纸未落。

## 具体内容

- [x] 全量重写 [05-pet-system.html](../../design/panels/05-pet-system.html),按规则阅读地图(身份→生命→工作→空间→互动→治理)重排展板:
  - **A 栖位 · 法定席位**:状态栏席位从品种名改为「豆豆 · IDLE」(规则 0 名字上屏),气泡改时段问候「早!」。
  - **B 主宠领养与命名**(规则 0):领养卡(七犬直选 + 交给缘分 + 名字井)+ 唯一/改名/换形象/老用户迁移 SPEC;显式标注 **「随机刷新」已废除**。
  - **C 七犬性格**(规则 D):品种架 + 性格参数表(眨眼/尾摆/打盹阈值/偏好微动作/口癖)。
  - **D 上下文状态机 + 活物引擎**(规则 C):五态机 + idle 微动作池六式(抓痒/舔爪/追尾转圈/竖耳/望屏外/伸懒腰)+ 专注保护/打扰预算。
  - **E 姿态审核定稿**(已落地):闭眼方案 A / 趴姿 / 探头 / Play 保留;SLEEP 阈值文案改为「按品种打盹阈值」。
  - **F 陪伴循环 · 共情**(规则 A/B/E/F):入场池四式 + 深夜哈欠 + 投喂四帧 + **工作共情六时刻** + 三帧律 + 羁绊档案。
  - **G 主动互动 · 互动记忆**(规则 G/H/I):摸头/叫名字/换窝安顿/久别亲近 + 玩具三件(球/骨头/小毯子,**非磷光暖色**)+ **重建右键菜单全集** + 互动记忆 SPEC + 记忆透明与重置。
  - **H 磷光通道**(规则 J):三空间动词 + 四标志场景(幽灵终端/Quick Look/浮层/分屏)实景接缝图 + 视觉纪律 + 节律降级 + `PetSpatialCue` 实现模型表。
  - **SPEC 治理层**:文案词表 + 事件优先级总表 + 节律预算 + 反规则 + 实现分期 P1–P5。
- [x] 删除全部「随机刷新」入口与「初始化随机抽取」表述,改为领养命名语义。
- [x] 扩展内嵌像素渲染器(与 `pet.rs` 同网格语义):新增 `headDx/headDy`(头部偏移)、`legDy`(腿行)、`flip`(镜像=转身)、`noShelf`;新增玩具非磷光画法 `ball/bone/blanket`;新增活物引擎/工作共情/主动互动姿态绘制。
- [x] 磷光通道四场景复用 SHEET 04 召唤线/残影、SHEET 06 scrim/浮层语汇,锚点依附真实 UI 接缝。
