# 任务: 小狗单击历史命令面板

## 背景与目的
深挖命令块（command block）功能，为小狗添加单击事件，点击时展示当前焦点面板的历史输出命令悬浮面板。点击其中命令后可实现终端滚动定位，且下方常驻命令条也联动更新定位。

## 任务详情
1. **PetView 改动**
   - 增加 `PetClicked` 事件，在 `on_mouse_up` 单击事件中触发。
   - 提供 `right()`, `bottom()`, `on_welcome()` 接口，供 Workspace 读取位置。
2. **TerminalView 与 BlockBar 改动**
   - 增加 `selected_block_id` 状态。
   - 增加 `select_command_block` 接口，进行绝对行滚动和命令块条覆盖。
   - 增加 `scroll_to_abs_line` 绝对行跳转方法。
   - 修改 `BlockBar` 渲染方法，支持覆盖历史 block。
   - 输入、滚动到最下方时自动重置覆盖状态。
3. **Workspace 改动**
   - 订阅 `PetClicked` 事件并管理 `pet_history_open` 状态。
   - 渲染绝对定位在小狗左侧的历史命令面板，带 Scrim 遮罩防穿透。
   - 逆序展示历史命令，绑定点击定位动作。
