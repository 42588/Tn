# 宠物球与骨头 Lottie 动画展示

## 定位

让宠物右键玩具菜单中的小球与骨头真正显示在当前 Lottie 宠物画面里,并在双击玩耍 `play` 段出现对应动画。

## 背景

已有 `Toy::Ball` / `Toy::Bone` 状态和右键菜单,旧的像素绘制路径里也有玩具摆放逻辑。但当前宠物主体已切到 `PetLottie::render_rgba_into` 输出 `RenderImage`,玩具需要进入 Lottie 渲染链路才会稳定显示。

本轮已按用户要求安装 `diffusionstudio/lottie` 的 `text-to-lottie` 技能到 `C:\Users\Gua\.codex\skills\text-to-lottie`。当前会直接按该技能要求使用仓库内 `design/pet-lottie` 官方 player 工程生成和验证动画资产。

## 具体内容

- 用官方 player 结构新增球与骨头的 Lottie 预览 scene。
- 生成运行时透明道具资产 `toy_ball.json` / `toy_bone.json`。
- Rust 侧根据 `state.toy` 选择对应道具 Lottie,与小狗主体同帧播放并 alpha 叠加。
- 保持右键“球 / 骨头 / 收起”的现有状态语义。
