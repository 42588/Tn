# BUG 发现清单处理

## 背景

用户在 [docs/BUG发现.md](../BUG发现.md) 报了 5 项。本文跟踪 triage 与修复。

## 状态

| # | 现象 | 状态 | 根因 / 方案 |
|---|---|---|---|
| 5 | QuickLook 滚轮等事件穿透到底层终端 | ✅ 已修(2026-06-12) | QuickLook 根节点与全部 7 处浮层 scrim(QL/命令面板/分屏/SSH 连接器/远端目录/布局/Agent 表单)统一 `on_scroll_wheel` 吞噬传播;浮层内滚动不受影响(子区先处理,根只拦"漏网") |
| 1 | SSH 会话执行命令不与宠物联动 | 待办(中型) | 宠物 Running/Success/Error 来自 OSC 133 shell-integration 事件;目前只对本地 pwsh 注入(`integrate_pwsh`),远端 shell 无 133 → 无块事件(同时也没有命令块条)。方案:远端 bash/zsh 注入 PROMPT_COMMAND/OSC 133 集成(连上后发注入串),或在 SSH 通道侧识别 prompt。与 #3 部分同根 |
| 2 | SSH 意外断开后快照不保留,重连即重开 | 待办(中型) | 自动重连走新 PTY,scrollback 清零。方案:重连复用同一 TerminalView/Terminal buffer(只换底层 PTY 流),断开期间冻结显示旧快照 + 顶部横幅;`tn-pty::ssh` 重连路径 + `terminal_view` 接管 |
| 3 | SSH 下 git init 后 Explorer 不实时展示文件变更 | 待办(中型) | 本地靠 notify watcher;远端无 watcher,`remote_git` 只在打开/手动刷新时拉取。方案:远端轮询(节流 `git status --porcelain`,会话活跃时 2-5s)或命令结束事件触发刷新(依赖 #1 的远端集成) |
| 4 | 幽灵启动器启动 agent 只能进默认目录,无法切换 | 待办(需设计) | 原型 SHEET 04 磁贴无目录维度。候选:a) agent 磁贴 ↵ 直启默认目录、`Shift+↵`/右键弹「最近目录」二级(与 WSL 下钻同源);b) 跟随主窗当前焦点 pane 的 cwd。需用户定夺后回写原型再实现 |

- 真机回归:QuickLook 打开后在浮层与压暗区滚动,底层终端 scrollback 不应再动(待用户复核)。
