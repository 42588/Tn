# 宠物 Lottie 动画全套接入与七品种

> 状态:已完成(2026-06-20)。承接「像素小狗矢量化重做(Lottie 探索)」原型,本轮**落地接入终端**并扩到全部品种。

## 目标

把像素宠物从 GPUI 逐帧重绘改为 Lottie 连续插值播放,像素身份不变(只平移/缩放不旋转),
全套动画 + 全部 7 个品种接入终端运行时。

## 渲染路径(已定夺)

- **不切 skia**。运行时由自写纯 Rust mini 播放器 [`crates/tn-ui/src/pet_lottie.rs`](../../crates/tn-ui/src/pet_lottie.rs)
  逐帧栅格化(矩形 + 填充 + 变换 + 线性关键帧 + 单层父子 + 解析式抗锯齿矩形覆盖),
  输出 RGBA→BGRA 喂 GPUI `RenderImage`;无 cmake/GPU 依赖。GPUI quad 直绘作回退。
- `design/pet-lottie` 下官方 skia 播放器仅作浏览器审稿。

## 动画(18 段,一条时间轴,每段打 marker;运行时按状态机跳段)

`peek / idle / typing / running / success / error / hover / click / play / drag / sleep /
feed / scratch / lickpaw / spin / stretch / lookout / earperk`

- **无缝循环**:所有循环段的子周期锁到「运行时循环体长度」(dur − LOOP_IN16)的整除数,
  跨循环点不跳帧。工作组循环体统一 120 帧(dur 136),spin 自带 80 帧体。
- **挤压拉伸**:success/play/feed 落地用 `rigSc`(Y)× `rigSx`(X)反向叠加做矮宽↔高瘦。
- **spin 重做**:旧版水平收放+镜像把像素挤成一条线(很丑)→ 改为绕地面小圈兜跑,
  远处抬高+缩小(`rigSc` 纵深),始终满宽、不翻转。
- **sleep**:站姿改姿态(收腿+眯眼+藏舌+呼吸),不切造型/不交叉淡入(淡入正是睡着闪烁根因)。

## 七品种(运动通用,逐品种出 JSON)

- **关键洞察**:运动(`osc`/`poseAt` 的平移/缩放量)对所有品种完全相同;只有像素网格、
  配色、部件锚点(眼/口鼻/舌/腿/尾)不同。
- 生成器 `script/gen_pet_lottie.mjs` 加 `BREEDS` 表(站姿网格/eyes/tail/tongue/face 衬底色,
  移植自 `pet.rs`;腿由底行双足自动切分),`build(CFG, breed)` 逐品种出
  `crates/tn-ui/assets/pet/<breed>.json`(westie/golden/shepherd/bichon/maltese/shihtzu/poodle)。
- 通用分层:base 轮廓 + 抬起的特征层(眼/口鼻/舌)+ 腿/尾;被抬起格在 base 补「面部衬底色」防露缝。
  无吐舌品种省 tongue 层,无尾品种(泰迪)省 tail 层。
- 运行时 `pet.rs::lottie_for(breed)` 选 JSON;构造时按 `breed` 装载,`pick_breed`/`adopt` 换品种时重载。
