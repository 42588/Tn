# 任务: 宠物双击不触发单击历史面板

## 背景与目的

宠物左键单击用于打开历史命令面板,双击用于触发玩耍互动。当前实现中,第一次点击的 `mouse_up` 会立即 emit `PetClicked`,第二次点击到来后才由 `mouse_down click_count >= 2` 识别双击,导致双击时历史命令面板先弹出,影响体验。

本轮目标是让单击进入短暂待确认状态,只有在双击窗口结束且没有第二次点击时才触发单击事件;双击到来时取消待确认单击并只触发玩耍。

## 范围

- 修改 `crates/tn-ui/src/pet.rs` 的单击/双击判定。
- 增加回归测试覆盖“双击取消待确认单击”。
- 不改变右键菜单、拖拽换窝、长按摸头、历史命令面板自身布局。
