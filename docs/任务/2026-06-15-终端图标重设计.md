# 终端图标重设计

## 背景
用户不喜欢当前的取景器图标风格，希望重新设计一个终端图标。经过询问，用户倾向于「现代磨砂玻璃 / 拟物化 (Glassmorphism / 3D)：半透明质感、模糊背景、立体光影，类似 macOS 现代图标」的风格。

## 方案与实现
修改 `scripts/gen_icon.py`：
- **底板 (Chassis)**：使用对角线渐变实现立体感 (BG_TOP #3B4358 -> BG_BOT #161B29)。
- **光效与边缘**：添加高光/阴影渐变构成的 1px 内发光效果，顶部额外绘制半透明的反光质感涂层（Glass Gloss）。
- **阴影 (Drop Shadow)**：图标整体加入高斯模糊效果的外阴影，使图标在不同背景上均有良好的立体剥离感。
- **图标主元素 (Mark)**：移除原本的四个取景器角标，改为经典的 `>` 提示符（白色）搭配磷光色块 `_` 光标（#5BE7C4）。对这两个元素附加了一层轻微的阴影以增强层次感。

## 验证结果
- 已运行 `python scripts/gen_icon.py` 成功生成新的 `tn.ico` 并在 `temp` 目录产生预览图 `tn_icon_preview.png`。
- 新图标覆盖尺寸完整 (16 到 256 px)，支持各级缩放显示。
