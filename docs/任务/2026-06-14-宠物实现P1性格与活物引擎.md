# 宠物实现 P1 · 性格系统 + 活物引擎

> 2026-06-14。把 SHEET 05 图纸(v2.3)与规则 `docs/宠物/宠物系统规则.md` 的
> **实现分期 P1** 落进 `crates/tn-ui/src/pet.rs`。这是 v2.3 唯一「无新网格、不需新审核稿」
> 即可实现的一期(板 C 七犬性格 + 板 D 活物引擎);P2–P5 引入的新帧(哈欠/饼干/玩具/
> 磷光裂缝)仍受「先审后做」约束,本期不动。

## 背景

`pet.rs` 此前停留在 v1 基座 + 板 E 已落地姿态(闭眼方案 A、趴姿、探头、双击 Play)。
图纸已推进到 v2.3,板 C/D 把「换一只狗」升级为「换一种陪伴感」,且明确
**零新网格、既有 dx/dy/镜像 变换可达**,故先做这一期。

## 本期落地(规则 C/D)

### 规则 D · 性格参数表(表即实现)
- 新增 `Personality` 常量表(七犬各一行),逐列对应图纸板 C SPEC:
  - `blink_min/max_ms` 眨眼间隔(西高地 4–6s 勤 … 西施 6–10s 眯)
  - `tail_amp` Idle 尾摆基础幅度(1/2/3px;泰迪 3px 最欢)
  - `sleep_after_ms` 打盹阈值(西施 45s 最爱睡 … 德牧 180s 站岗)
  - `bark` 口癖气泡(德牧 = 空串 → 不出声,单击/Play 以**点头 1 格**代替)
  - `micro_weights[6]` 微动作偏好权重
- 替换原先硬编码:眨眼 4–8s、打盹 `SLEEP_AFTER_MS=90s`(常量删除)、Idle 尾摆 1px、
  单击「汪!」/Play「汪汪!」→ 全部走品种参数。

### 规则 C · 活物引擎(idle 微动作)
- 新增 `Micro` 六动作池(顺序对齐权重表):抓痒 / 舔爪 / 追尾转圈(水平镜像)/
  竖耳听声 / 望屏外 / 伸懒腰。
- 调度:仅纯 Idle、无探头、无气泡时,**20–60s 随机一个**,时长 ≤1.5s 回 idle;
  按品种性格加权选取、**同一动作不连续两次**、**打扰预算 ≤2 次/分钟**。
- 专注保护:离开 Idle 立即弃帧(`micro=None`);reduced motion 整章停用。
- 渲染:全部复用既有 `dx/dy` 变换;追尾 = 整身 `x→13-x` 镜像(翻转即转身);
  无新网格、无新原语。
- 80ms 微动作驱动(抓痒抖动 / 追尾镜像快于 240ms 主 tick),到点清帧自停。

## 验证

- `cargo test -p tn-ui --lib pet` → 4 绿(新增 `personality_table_is_sane`、
  `earless_breeds_dont_perk_ears`;原 `sprites_are_well_formed`、`lie_pose_hugs_the_shelf` 不破)。
- `cargo clippy -p tn-ui` 未引入新告警(pet.rs 余留 2 条为既有代码:相位翻转同块、菜单闭包复杂类型)。
