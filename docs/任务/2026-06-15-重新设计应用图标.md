# 任务: 重新设计应用图标

## 缘起与目标
当前 `scripts/gen_icon.py` 生成的 `tn.ico` 样式较为单调(仅有黑底、四个磷光角标和中心方块),不符合「磷光 Phosphor」设计语言的高端精密仪器质感。
目标是重新设计图标生成脚本,使其生成的图标展现出精密仪器舱体、阴极射线管(CRT)屏幕栅格、终端提示符 `> █` 以及各类微型仪表面板细节,提升视觉品质,达到 WOW 的高级感。

## 视觉设计构想 (v2: 科技跃动风格)
1. **多层机箱底座 (Chassis)**:
   - 舱体采用深邃且动感的暗色渐变（从 `#22263F` 到 `#0E0F16`）。
   - 圆角半径比例缩减为 `0.20`（原 `0.235`），使舱体更加精美干练。
   - 舱体外轮廓描绘一圈具有微弱霓虹发光质感（Neon Glow）的磷光边框，使用双通道绘制法实现。
2. **渐变显示屏幕 (Vibrant Screen)**:
   - 屏幕背景采用更具科技质感的深蓝色至深紫色渐变（从 `#121625` 到 `#080A10`），让光标在深色背景中显得更加立体。
   - 屏幕边缘进行柔和过渡，增强空间维度感。
3. **双通道发光角标 (Glowing Viewfinder)**:
   - 角标尺寸精细化为 `w = 0.035 * s`，折角更加纤细。
   - 采用双通道发光渲染法：底层以宽线和低透明度（45 alpha）绘制发光阴影层，顶层以标准宽度和高亮绘制实线层，实现荧光屏的自然发光质感。
4. **双通道发光光标 (Glowing Block Cursor)**:
   - 保留品牌核心的取景器中心方块光标，比例优化为 `[0.44 * s, 0.42 * s, 0.56 * s, 0.58 * s]`，同样使用双通道发光绘制法，使其在屏幕正中央呈现明亮、微弱发光的立体效果。

- **功能验证**: 运行 `python scripts/gen_icon.py` 成功生成了最新的 `crates/tn-ui/assets/tn.ico` (372,526 字节)。
- **视觉验收**: 预览文件 `tn_icon_preview.png` 显示：
  1. 舱体外沿环绕着一层由双通道（宽透明阴影 + 窄高亮边线）渲染出的绿色霓虹发光（Neon Glow）描边，展现出非常亮眼的未来科技质感。
  2. 屏幕区域和舱体本身均使用了流畅的对角线渐变（从 `#22263F` / `#121625` 渐变至深色），色彩层次极其丰富。
  3. 四角取景器折线和中心块状光标通过双通道发光渲染，具备逼真的 CRT 荧光屏溢光质感，极具科幻冲击力。
  4. 16px 尺寸显示极为干净且高对比，发光边框与中心块状光标提供了完美的微缩辨识度。
  5. 优化了 ICO 打包性能并完全消除了 DeprecationWarning。
