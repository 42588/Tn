# 任务: 重新设计应用图标

## 缘起与目标
当前 `scripts/gen_icon.py` 生成的 `tn.ico` 样式较为单调且扁平，视觉细节杂乱堆砌，不符合「磷光 Phosphor」设计语言的高端精密仪器质感。
目标是重新设计图标生成脚本,使其生成的图标展现出极具物理厚重感的 3D 等轴测终端舱体底座、下沉屏幕反应舱、悬浮发光的 3D 晶体（等轴测蚀刻 `T` 和 `n`）等精美细节，达到真正 WOW 的高级视觉品质。

## 视觉设计构想 (v4: 3D 阳极氧化钛·等轴测悬浮激光蚀刻晶体舱)
1. **钛合金等轴测机箱底座 (3D Titanium Isometric Chassis)**：
   - 底座拉高厚度，左右面使用垂直的阳极氧化钛金属渐变色。
   - 侧壁加工双散热格栅凹槽，交界处以 1.5px 宽的半透明白光勾勒出极精细的 CNC 棱边高光。
2. **下沉式微光屏幕舱 (Recessed Basin)**：
   - 顶面往内收缩形成微凹的屏幕舱盆地，底面采用深邃对角渐变（`#0F111C` -> `#030406`），四周反射荧光溢光。
3. **悬浮 3D 激光蚀刻晶体 (Laser-Etched Suspended Core)**：
   - 反应舱上方悬浮一颗 3D 半透明玻璃晶体，底部投射椭圆绿发光阴影，产生强烈的浮空悬浮物理质感。
   - 晶体顶面高亮，**左、右侧壁在等轴测投影下分别用双通道激光长短线蚀刻出 `T` 和 `n` 字母**（发光包裹白芯）。
4. **定位角标与粒子点缀**：
   - 四角浮空 3D 倾斜定位线框，散落微小高亮科技点，提升设计信息密度。
5. **小尺寸自适应过滤 (LOD)**：
   - 在 16px - 32px 极小图标尺寸下，自动隐藏细线和角标，仅保留底盘轮廓和 3D 发光晶体块，保证在任务栏/桌面极其干净锐利。
