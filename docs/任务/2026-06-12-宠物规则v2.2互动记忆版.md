# 宠物规则 v2.2 互动记忆版

## 定位

记录在既有 [宠物系统规则](../宠物/宠物系统规则.md) v2.1 基础上继续升级主动互动深度的文档任务。

## 背景

用户要求继续迭代 `docs/宠物/宠物系统规则.md`。经确认,本轮重点不是日常循环或工程拆解,而是**互动深度**。初版推荐轻重适中的亲密互动包后,用户明确偏好重型方案;最终定夺为 v2.2「互动记忆版」。

## v2.2 定夺

| 主题 | 规则 |
|---|---|
| 互动范围 | 主动触发优先:摸头、叫名字、换窝安顿、玩具摆设和久别亲近 |
| 记忆范围 | `favorite_interaction`、`favorite_perch`、`toy_choice`、`quiet_hours_seen`、`last_affection_at` 五项 |
| 记忆用途 | 只改变演出权重和收尾细节,不生成任务、不解锁内容、不影响健康/心情 |
| 透明机制 | 右键菜单显示最多 3 个短标签,并提供「重置互动记忆…」 |
| 隐私边界 | 不扫描终端输出、不读取文件/路径/窗口标题、不记录完整事件日志 |
| 反规则 | 不做行为画像、不做隐藏亲密度分数、不把记忆变成提醒 |
