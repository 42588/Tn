# 任务: 幽灵终端 Agent 改动预览修复

## 背景与目的

幽灵终端启动器启动的 agent 能在右侧活动栏显示“本次改动”,但点击改动文件后无法弹出 QuickLook 预览。主工作区的 agent pane 已经通过 `OpenInQuickLook` 事件接入 QuickLook;幽灵终端同样嵌入 `TerminalView`,但缺少自己的 QuickLook 宿主和事件订阅。

本轮目标是在幽灵终端内补齐 QuickLook 预览通道,让右侧改动卡片点击后打开 Diff 预览,并保留本次改动列表内的上下导航。

## 范围

- 修改 `crates/tn-ui/src/quick_terminal.rs`。
- 复用现有 `QuickLook` / `OpenInQuickLook` / `RailFileTarget` 机制。
- 不改变主工作区 QuickLook 行为、不改变活动栏 diff 计算。
