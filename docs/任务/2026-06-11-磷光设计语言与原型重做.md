# 磷光设计语言与原型重做

## 定位

本文记录用「磷光 Phosphor」设计语言全量重做 `design/` 原型、取代 Calm Glass 的任务过程与验证结果。

## 背景

「原型截图样式逐页复原」任务多轮真机回归后确认:旧 HTML 原型(Calm Glass)依赖 backdrop blur + noise 抹平大面积透明渐变,GPUI 无逐元素 blur 与噪点,大面积玻璃在 8-bit 下必然色带 —— 视觉目标物理性不可达。用户决定:重新设计一套未来感、高级、且能被 Rust 精准还原的原型,覆盖旧设计风格;三大特色(QuickLook 编辑器、幽灵终端、宠物系统)各占一页。

## 设计要点(供实现参考)

- 海拔:L0 `#0B0E16` → L4 `#232C42` 全不透明;发丝线白 5/9/16%;磷光 `#5BE7C4` 唯一生命色。
- 深度:平铺=2px 接缝+1px 边、零投影;浮层=L3+h2 边+外投影+纯色 scrim。
- 焦点:四角角标 + ph-dim 描边,无 glow;选中/状态统一左 2px 脊。
- 圆角 10/6/4/3;标题栏 42、状态栏 26、plate-head 34(agent 38)、活动栏 248。
