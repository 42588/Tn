## 设计史决策

- 2026-06-11:Calm Glass 玻璃方案废弃。根因:HTML 原型依赖 backdrop blur + noise 抹平大面积透明渐变,GPUI 无逐元素 blur 与噪点,大面积玻璃必然色带(真机截图证伪,详见 [图形界面与渲染坑位](共享记忆/图形界面与渲染坑位.md))。
- 2026-06-11:启用「磷光 Phosphor」。核心转向:从"模仿材质"转为"从 GPUI 渲染能力出发设计",原型 CSS 与 GPUI API 一一对应,原型即实现规格。
