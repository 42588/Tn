# 组件规格与自动生成数值

## 定位

本文负责承载设计令牌、组件规格和新组件装配速记中的确切数值。这里是实现时查数值的权威落点，母文档不再承载大段规格表。

## 背景

组件规格由 `TN_GEN_SPEC=1 cargo test -p tn-ui spec_gen` 从 `design/calm-glass.css`、`tn-dark.toml` 和 `style.rs` 生成或校验。前置章节讲译法 HOW，本文讲确切数值 WHAT。

## 具体内容

#### 新组件装配速记(数值见本文件自动生成组件规格,结构见对应 panel)

- **活动栏 `.arail`**(agent 右栏,[panels/02](../../design/panels/02-workspace-panes.html)):agent body 用 `flex().flex_row()`,内放 `终端流(flex_1 + min_w(px(0.)))` + `.arail`(`flex_none().w(px(212.))` + 左 1px 分隔)。栏内 `.astat` 状态行 + `.achip` 改动卡(`.afile` 文件名 / `+N −N` + `.adiff` 行;当前文件 `.cur` 带 accent 内 ring)。**数据来自 `git diff` + JSONL,不解析终端正文**(见 [智能体用量与诚实原则](../产品体验/智能体用量与诚实原则.md))。
- **速览浮层 `.quicklook`**([panels/03](../../design/panels/03-side-panels.html)):贴树右缘的 absolute 浮层(层级高于终端),bg 比常驻面板更实(~0.9 alpha 压住后面终端);头 `.vh` + 正文 `.code`(行号 `.ln` / 增删 `.cl.add/.del`)+ 底 `.qlfoot`;左缘 `.seam` accent 竖线。真机宜独立 popup/overlay 层。**两态键位**:预览态 ↑↓ 换文件、编辑态方向键归编辑器(见 [快速预览与编辑](../产品体验/快速预览与编辑.md))。
- **app 菜单 `.appmenu`**([panels/01](../../design/panels/01-window-chrome.html)):logo 下 absolute 下拉,`.mi` 项(`hover` 高亮、`.danger` 红)+ `.sep` 分隔;`.brand .caret` 提示可点。logo 改可点后**拖窗挪标签条空白**(见 [窗口框架与面板映射](../产品体验/窗口框架与面板映射.md))。
- **欢迎 launchpad `.welcome`**([panels/05](../../design/panels/05-states.html)):居中列 = 磁贴 `.tiles/.tile` + `.recent`(最近目录,色点 = 会话类型)+ `.whints`(快捷键提示);默认新窗/新标签。
