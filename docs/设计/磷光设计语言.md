# 磷光设计语言

## 定位

本文是「磷光 Phosphor」设计语言的权威规范:定义原则、token、渲染契约和实现映射。视觉原件以 `design/phosphor.css` 与 `design/panels/*.html` 为准,本文负责"为什么"与"怎么落地"。

一句话:**精密仪器舱体 + 磷光生命色 —— 结构给秩序,磷光给生命。**

## 为什么取代 Calm Glass

| 维度 | Calm Glass(已废弃) | 磷光 Phosphor |
|---|---|---|
| 材质来源 | 半透明玻璃渐变 + 高光洗层 | 不透明海拔阶梯 L0→L4 |
| HTML 依赖 | backdrop blur + noise 抹平色带 | 无 blur、无噪点、无大面渐变 |
| GPUI 还原 | 不可达(8-bit 大面渐变必色带,真机证伪) | 一一对应,原型即规格 |
| 深度 | 投影 + 透明层叠(平铺投影渗血) | 平铺=接缝,浮层=投影 |
| 强调 | 高光、边缘导光 | 磷光只给活物,零发光 |

## 四原则

1. **不透明海拔**:大面积只用纯色阶梯;海拔差就是材质感。
2. **接缝即深度**:平铺板面零投影,2px 接缝露出底盘 + 1px 发丝边;投影只属于真浮层(QuickLook、命令面板、菜单、幽灵终端、确认件、Toast)。
3. **磷光纪律**:`#5BE7C4` 是唯一生命色,只出现在光标、运行态、焦点角标、选中脊、活动数据;永不做装饰发光。Agent 身份色(紫/蓝)只出现在头像、tab 棒、用量环。
4. **像素公民**:宠物(8px 像素网格)与幽灵标和终端网格同源,逐 quad 直绘;趣味组件不破坏专业纪律(不入 grid、不抢焦点、可关闭)。

## 渲染契约

原型 CSS 只使用 GPUI 已验证能力:

| 原型用法 | GPUI 落点 |
|---|---|
| 纯色填充 / alpha 叠色 | `Div::bg(Rgba)` / `cola()` |
| 1px solid 边 | `border_1().border_color()` |
| 圆角 ≤10px | `rounded(px(..))` |
| `linear-gradient` ≤4px 条带(tab 身份棒、幽灵顶缘、召唤线) | `linear_gradient()` 小元素 |
| `box-shadow` 仅浮层外投影 | `BoxShadow`,父级不裁剪 |
| 纯色 scrim `rgba(5,7,12,.62)` | absolute 全幅 Div |
| 用量环(HTML 用 conic 示意) | canvas path 弧线 |
| 像素狗 / 幽灵标 | 逐像素 quad 直绘(≤168 quad/帧) |
| 焦点四角角标(11×2px 双臂) | 4 个 absolute 小 Div |
| **禁用** | backdrop-filter / filter / inset shadow / text-shadow / 大面渐变 / 噪点 / letter-spacing / radial·conic(环除外) |

## 签名部件

- **焦点角标**:取景器式四角磷光标记 + 1px ph-dim 描边,替代 glow 焦点。
- **身份棒**:active tab 顶部 2px 横向小渐变,色=会话所有者(磷光=shell,紫=Claude,蓝=Codex)。
- **状态脊**:命令块/选中行/磁贴左缘 2px 竖棒,全系统统一"选中/状态"语法。
- **接缝**:canvas `gap:2px` 露出 L0,板面各带 1px h0 边。
- **残影**(幽灵终端专属):窗后 1–2 道内收下沉的 1px ph-dim 轮廓,无填充无模糊。
- **岗台**(宠物专属):1px h1 发丝线 + 左 28px ph-dim 点睛,宠物世界与 UI 的唯一接缝。
- **块光标**:磷光填充 + ph-ink 反相重绘字符;CJK 盖双格;选区逐单元格 ph-soft。

## 实现映射与验收

- token 落点:`crates/tn-ui/src/style.rs` 单源;各页 SPEC 表给出元素级规格,照抄即可。
- 验收:真机 `target/ui-shots` 截图对照图纸,核对三件事 —— 海拔阶梯、接缝深度、磷光纪律;不引入样式守卫测试。
- 渲染新坑先写 [图形界面与渲染坑位](../共享记忆/图形界面与渲染坑位.md),再回写本契约。
