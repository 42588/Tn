# 快速预览与编辑

## 定位

本文记录 Quick Look 的用户可见体验:浮层定位、预览/编辑两态、Diff 审阅、保存安全、光标与 CJK 选择语义。实现细节和修复复盘不在本文展开。

## 背景

Quick Look 是点文件树或在树焦点按 `Space` 弹出的玻璃浮层。它贴文件树右缘、浮于终端、不占分屏,目标是快速看一眼或顺手小改,不是替代重度编辑器。真要大改仍应回终端使用 vim 或后续 Editor Pane。

## 两态交互

| 态 | 键 | 行为 |
|---|---|---|
| 预览 | `Up` / `Down` | 换文件,实时跟随文件树选中 |
| 预览 | `Tab` | 切换 File / Diff |
| 预览 | `Enter` | File tab 进入编辑态;Diff tab 跳到对应 File 行 |
| 预览 | `PgUp` / `PgDn` | Diff tab 跳上/下一个 hunk |
| 预览 | `Esc` / `Space` | 收起浮层,焦点还给终端 |
| 预览 | 鼠标拖选 + `Ctrl+C` / `Ctrl+A` | 只读复制,不可修改 |
| 编辑 | 打字、`Backspace`、`Delete`、`Enter`、`Tab` | 编辑文本,`Ctrl+S` 写盘 |
| 编辑 | 方向键、`Home`、`End`、`PgUp` / `PgDn` | 移动编辑器光标,不再换文件 |
| 编辑 | `Shift` + 方向、鼠标拖选 | 选区,语义是“实心块拖到哪就选到哪” |
| 编辑 | `Ctrl+C/X/V`、`Ctrl+Z/Y`、`Ctrl+A` | 复制剪切粘贴、撤销重做、全选 |
| 编辑 | `Ctrl+F` / `Ctrl+H` | 查找 / 替换 |
| 编辑 | `Esc` | 回预览 |

## 体验决策

- 光标使用终端式就地反相块,瞬时、精确、随字符列宽变化:中文 2 列宽,英文 1 列宽。曾试过平滑滑动光标,owner 实测卡顿和字飘后撤回;文本编辑器的光标优先级是准而不是漂。
- CJK 精确依赖固定单元格渲染,列到像素严格对齐,避免中文行的光标、选区和横向滚动漂移。
- Diff 审阅保持在浮层内完成。Diff tab 支持鼠标拖选、`Ctrl+C`、`Ctrl+A`;`PgUp` / `PgDn` 在 hunk 间跳转;`Enter` 或双击当前 diff 行切回 File tab 并定位到对应的新文件行,目标行用短暂 accent 高亮。
- 删除行定位到同一 hunk 内最近可对应的新文件行,等距时偏上一行。远端 diff 只在当前 Quick Look 内切 File,不把远端路径当本机路径打开;该路径仍需 SSH 实机复验。
- 本地文本保存记录打开时的编码、换行风格、mtime、size 和 hash。`Ctrl+S` 前若磁盘文件被外部修改或删除,先显示冲突条“重新载入 / 取消 / 覆盖保存”,默认不静默覆盖。
- 正常保存保留原 UTF-8 BOM、UTF-16、GBK 与 LF/CRLF 风格。
- 长行可横向滚动,底部使用自绘细滚动条;编辑态光标会跟随横滚。预览每次从行首开始,不残留横滚位置。
- 浮层下铺 click-away scrim,故意不盖文件树、标题栏、状态栏;点空白收起。面板根吞点击,避免穿透到终端。

## 状态

- 预览、编辑、只读选择、Diff 选择、Diff 跳转、本地保存守卫已落地。
- 散文或 Markdown 长段落目前只能横向滚动,软换行属于后续编辑器能力。
- 远端 Diff 路径和远端 guarded write-back 需继续按真实 SSH/SFTP 场景复验。
