# 窗口框架与面板映射

## 定位

本文记录窗口框架、标题栏、状态栏、app 菜单和原型面板到真实实现的用户可见映射。具体样式数值和 crate 实现细节分别归界面样式与系统架构文档。

## 标题栏与状态栏

- Tn 使用自绘集成标题栏。
- OS 仍负责拖窗、最小化、最大化和关闭的非客户区命中,避免双触发。
- 状态栏显示分支、会话数、文件类型、编码、主题和焦点 pane 用量等多段信息。

## app 菜单

- 左上 `Tn` 品牌是可点击菜单入口,带 caret 提示。
- 品牌从“拖窗把手”改为“可点按钮”,拖窗改靠标签条空白区域。
- 这是有意取舍:logo 当菜单入口比当拖把更有用。
- 菜单点外部 scrim 关闭。
- 菜单项接真实动作:新会话、分屏启动器、新标签欢迎页、打开文件夹、布局存取、文件浏览器开关、设置、主题、重载配置、关于和退出。
- “打开文件夹”只影响当前焦点 pane;SSH pane 不弹本机 picker,而是打开当前远端 root 的应用内 SFTP 浏览。
