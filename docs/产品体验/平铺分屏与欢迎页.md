# 平铺分屏与欢迎页

## 定位

本文记录 Tn 的平铺分屏、标签、欢迎页、应用内添加 Agent、命令面板和快捷键体验。具体配置写入、crate 边界和持久化实现见架构或新增模块文档。

## 平铺与标签

- 分屏采用 n-ary 平铺模型,支持任意方向切分。
- 键盘可调整窗格尺寸,分隔线支持拖拽。
- 分隔线拖拽使用 commit-on-release:拖动时只移动预览线,释放时才 resize 一次,避免 ConPTY 在每帧 resize 时抖动。
- 默认布局是 `Explorer(窄) | Agent(大) + 小 shell`。

## 欢迎页

- 新标签或首个标签显示欢迎 launchpad。
- launchpad 提供默认通用 `Agent` profile、shell、WSL、SSH 等启动入口。
- 点磁贴直接启动会话,普通用户不需要先编辑 `config.toml`。
- 从 shell、agent 或 SSH pane 新建 welcome 标签时,左侧 Explorer 回到 Windows 默认 Host 目录,优先 `USERPROFILE`,没有时用 `HOME` 兜底。
- 新 welcome 标签不继承上一个 pane 的远端 root、展开态或选中状态。
- 如果用户在 welcome 页手动“打开文件夹”,该选择只作为当前 welcome 页下一次启动会话的 cwd。

## 应用内添加 Agent

- launchpad 尾部提供“+ 添加 Agent”磁贴。
- 点击后显示居中的玻璃浮层表单:名称、命令、颜色预设、“由 Agent 自绘光标(Ink TUI)”开关和实时预览。
- 保存后立即生成磁贴,无需重启,也无需手编配置文件。
- 自定义磁贴 hover 时提供编辑和删除入口。
- 名称字段支持中文输入。
- 基础档是通用 agent:托管终端加活动栏,没有用量环,因为没有真实遥测。
- 当命令是 `claude` 或 `codex` 时,Tn 使用内置日志解析器显示真实用量环,保留用户选择的颜色和名称,不需要额外配置。
- 进阶“用量遥测”字段面向开发者:可填一个输出 JSON 用量的伴随程序。若勾选联网,启动前必须弹确认卡,默认拒绝,确认后才连接。

## 命令面板与快捷键

- `Ctrl+Shift+P` 打开命令面板。
- `Ctrl+Shift+B` 切换侧栏。
- `Ctrl+Shift+J` 收起 Quick Look。
- `Ctrl+Shift+*` 在中文或多布局 Windows 中可能被系统或 IME 吞掉,真机不可达时应改键。
- 命令面板中文搜索暂 parked:IME 合成只经 `EntityInputHandler`,面板未接入,当前只搜索英文。
