### 里程碑摘要

M0-M5 均已落地。owner 实际执行顺序为 M0 -> M1 -> M3 -> M4 -> M5 -> M2。当前实现细节以代码、发布记录和专题文档为准。

| 里程碑 | 交付摘要 | 状态 |
|---|---|---|
| M0 骨架 | 工作区、工具链、cargo-deny、GPUI 窗口、`tn-core`、`LocalPty`、`TerminalView` | 已落地 |
| M1 可日用本地终端 | 每格颜色、Tab、n-ary 分屏、键盘改尺寸、分隔线拖拽、滚动/选择/复制粘贴、输入层、push+vsync、`tn-config` | 已落地 |
| M2 WSL + 远程 Linux | WSL 端到端、SSH `SshBackend` 密钥路径真机端到端 | WSL 已落地 / SSH 密钥路径真机过 |
| M3 shell 集成 + block | `tn-shell` 旁路 OSC 解析、`tn-blocks` 状态机、Warp block 底栏和 alt-screen 隐藏 | 已落地 |
| M4 AI 托管 + 命令面板 + 颜值 | `tn-ai` 用量、agent 检测、命令面板托管、Calm Glass UI、面板逐组件对齐 mockup | 已落地,颜值持续微调 |
| M5 Quick Terminal | 全局热键、无边框置顶 PopUp、边缘滑入、失焦自动隐藏、启动器、紧凑 SSH 连接器 | 已落地,真机肉眼验待持续 |

### 原型同步轨道

当前活跃轨道是把 `design/panels/` 的定稿原型逐步端口到 GPUI:

- Quick Look 速览浮层。
- 欢迎 launchpad。
- Agent 活动栏。
- app 菜单。
- per-pane 工作区上下文。
- SSH 连接体验。

规则:

- 设计真源是 `design/mockup.html`、`design/panels/` 和 `design/calm-glass.css`。
- 样式实现遵守 [界面样式实现规则](../界面样式实现规则.md)。
- 新可视件先在 mockup/panels 出稿,再落 GPUI。
- 守卫包括 token drift、roots mirror、no hardcoded、spec gen 等。

### 开发环境

- Rust stable。
- 目标 `x86_64-pc-windows-msvc`,由 `rust-toolchain.toml` 固定。
- 需要 VS C++ Build Tools + Windows SDK,GPUI/DX 必需。
- `cargo` 常见路径:`%USERPROFILE%\.cargo\bin\cargo.exe`。
- dev 构建使用优化依赖的配置,避免 alacritty advance、snapshot、row_runs 和 GPUI 布局/paint 在 opt0 下把高频 TUI 体验拖慢。

### 常用命令

```powershell
cargo build --workspace
cargo test --workspace --lib
cargo test -p tn-core
cargo test -p tn-ui --lib
cargo run -p tn-cli
cargo run -p tn-cli -- wsl.exe -d Ubuntu -- echo HELLO_TN_MARKER
cargo run -p tn-app
$env:TN_AUTOQUIT="1"; cargo run -p tn-app
$env:TN_DEMO="1"; cargo run -p tn-app
$env:TN_PERF="1"; cargo run -p tn-app
cargo bench -p tn-core --bench engine
```

根 `.cargo/config.toml` 还提供 `cargo smoke`、`cargo lint`、`cargo t` 等别名。

### 开发约定

- `gpui` 只准出现在 `tn-ui` / `tn-app`。
- headless crate 必须能单测。
- 不要把 alacritty 类型泄漏出 `tn-core` 边界。
- `LocalPty` 是本地进程环境边界,负责 spawn、IDE 环境变量剥离和 PATH shim 合并。
- 改依赖版本走 `[workspace.dependencies]`。
- git 依赖必须钉死 `rev=`。
- 行尾 LF。
- 若用 AI 协作提交,提交信息结尾带 `Co-Authored-By` 行。

### 验证清单

- M0:`cargo run -p tn-app` 开窗、跑 PowerShell、`dir` / `vim` / `cat 大文件` 不明显卡顿、resize 重排;`TN_AUTOQUIT=1` dump 出含命令输出的网格;`cargo test -p tn-core` 绿。
- M1:Tab、分屏、滚动、复制粘贴、配置、主题可用;`Ctrl+Shift+R` 热重载颜色。
- M2:开 WSL Tab 与 SSH Tab,各跑 `htop` / `vim`;SSH 空闲过 keepalive 仍在;断网/断连 UX 干净。
- M3:PowerShell 集成注入后,每条命令形成 block;进 `vim` 时 block chrome 消失,退出恢复;`tn-shell` 用录制 OSC 流单测。
- M4:`Ctrl+Shift+P` 拉起 agent,形成带实时状态条的 agent surface;最近命令面板可重跑已验证 block。
- Quick Look / Editor:覆盖本地/远端打开、编辑、保存冲突、Diff 跳转、CJK 命中、IME、查找替换和大文件边界。
- SSH / Remote:覆盖密钥、密码、TOFU、错误卡、断线、ssh-config alias、远端目录 picker、远端保存和远端 hunk。
