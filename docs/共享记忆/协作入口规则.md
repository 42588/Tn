## 具体内容

- `AGENTS.md` 是 Codex 的项目入口。
- `CLAUDE.md` 是 Claude 的项目入口。
- 两者只保留启动时必须读取的索引和最小规则,不复制长篇说明。

- `docs/共享记忆/踩坑记录.md` 是当前唯一保留的长期历史记忆。
- 新一轮设计开始前必须先读取踩坑记录,但不得恢复旧方案、旧路线图或旧任务文档。
