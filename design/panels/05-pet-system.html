J1 · 三个空间动词 — 一切通道演出由这三个动词组合而成
开门
宠物打开隐藏空间:真实 UI 接缝(岗台线/屏顶召唤线/QuickLook 边缘/pane 分隔线)亮起 1px 磷光裂缝,宠物蹲下蓄力后钻入或跃入。
边界:只服务关键转场,不绑普通 hover/button。
穿梭
宠物在 UI 空间之间迁移:主终端消失 → 目标 UI 边缘探头/落地,保留同一只主宠身份。
边界:不复制宠物;同一时刻只有一个主宠位置。
召回
UI 收起,宠物带着空间闭合:回头叼/拉一个 磷光光点,面板或幽灵窗收束,从原岗台探头回来。
边界:UI 收起不等待动画完成,动画可追随收尾。

视觉纪律(J3)
· 魔法只由 磷光线 · 裂缝 · 光点 · 残影 组成;禁烟雾 / 粒子风暴 / 大面渐变 / 柔光雾 / 全屏遮罩。

· 宠物仍是像素小狗:不长翅膀、不变形、不变光团;消失/出现靠遮挡、位移、裂缝。
· 颜色只用 ph / ph-dim / h2;亮度峰值 ≤ 幽灵终端顶缘,不压过正式焦点。
· 完整共演只给标志性入口;普通面板开关、菜单 hover、状态更新不使用通道语法。

· 主动演出(非用户触发):微动作 ≤2 次/分钟 · 庆祝 ≤2 次/10 分钟 · 气泡 ≤4 条/小时。
· 用户主动互动不占预算,但同类 2s 内合并;叫名字冷却 10s;久别亲近每天最多 1 次。
· 空间共演:完整 ≥30s 间隔,10–30s 内重复降短版,10s 内连续操作静默换岗。
