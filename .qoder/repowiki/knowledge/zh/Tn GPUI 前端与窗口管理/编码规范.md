- 所有 UI 颜色必须通过 `style::col()` 或 `cola()` 辅助函数引用主题令牌，禁止在视图代码中硬编码与主题色匹配的 RGB 值。
- 涉及耗时 IO 或计算的任务（如 Git 状态查询、AI 用量轮询）必须卸载到 `cx.background_executor()` 或独立线程中执行，避免阻塞 GPUI 主渲染循环。
- 视图组件的复杂后台逻辑（如 PTY 读取、光标闪烁、退出监控）被拆分到独立的 `impl` 模块（如 `terminal_view/io.rs`）中，以保持主视图文件的渲染逻辑清晰。
- 使用 `WeakEntity` 在异步任务中引用 UI 组件，并在更新前检查 `.update()` 的返回值，以安全处理组件已销毁的情况。
