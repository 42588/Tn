- **核心架构**：采用 `gpui` 声明式 UI 模型，以 `Workspace` 为根实体管理多标签页（Tabs）和 N 叉树状布局（`LayoutNode`）。
- **渲染层**：通过 `TerminalView` 封装 PTY 后端，利用 `BlockModel` 解析 Shell 集成协议（OSC 133/633）实现命令块视图。
- **样式系统**：`style.rs` 作为单一真源，定义 Calm Glass 玻璃拟态设计令牌（颜色、圆角、阴影），并通过 `spec_gen` 测试确保与设计稿同步。
- **平台适配**：`platform.rs` 处理 Windows 专属逻辑，包括全局热键注册、系统托盘图标及 IME 输入法修复。
- **状态管理**：使用 `Arc<Mutex<T>>` 共享 PTY 读写状态，结合 `cx.spawn` 异步任务处理后台轮询（如 AI 用量、Git 状态）。
