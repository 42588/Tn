- **分层解耦设计**：采用 `tn-core` (引擎) + `tn-pty` (后端) + `tn-ui` (前端) 的分层架构，确保核心逻辑 Headless 可测且与 GPU 渲染隔离。
- **统一 PTY 抽象**：定义 `PtyBackend` Trait 统一本地 ConPTY、WSL 及 SSH 后端的同步读写接口，支持动态切换会话类型。
- **旁路语义解析**：在字节流进入网格前通过 `tn-shell` 进行 OSC 133/633 旁路解析，驱动 `tn-blocks` 状态机实现命令块级 UI 增强。
- **异步渲染管线**：利用 GPUI 的 vsync 帧时钟合并重绘请求，结合 `generation` 缓存机制与后台 Grid 序列化，实现高频输出下的零卡顿渲染。
